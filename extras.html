Extras & Resources

Additional tools and resources for power users.

  • Everything File Search
    https://www.voidtools.com/downloads/
    Instant Windows file indexing and searching.
  • Public APIs Directory
    https://github.com/public-apis/public-apis
    Large curated list of free APIs.
  • MSI Afterburner
    https://www.msi.com/Landing/afterburner/graphics-cards
    GPU monitoring and overclocking utility.
  • VirusTotal
    https://www.virustotal.com/gui/home/upload
    Scan files with multiple antivirus engines.
  • WizTree (Disk Analyzer)
    https://diskanalyzer.com/download
    Visual storage usage breakdown tool.
  • Temp-Mail
    https://temp-mail.org/
    Disposable email addresses for signups.
  • Have I Been Pwned
    https://haveibeenpwned.com/
    Check if email appears in data breaches.
  • SocialEye
    https://socialeye.net/
    Scan exposed personal data online.
  • Remove Paywall
    https://www.removepaywall.com/
    Bypass soft paywalls for articles.
